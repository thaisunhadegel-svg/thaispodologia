# 📱 Podologia App

Um aplicativo web progressivo (PWA) para gerenciar pacientes de podologia com suporte offline.

## 🎯 Funcionalidades

- ✅ **Adicionar Pacientes** - Cadastre nome, contato e queixa
- ✅ **Armazenamento Local** - Dados salvos no navegador (localStorage)
- ✅ **Suporte Offline** - Funciona sem conexão com internet
- ✅ **Instalável** - Install como app nativo em Android, iOS e Desktop
- ✅ **Interface Responsiva** - Design adaptado para todos os tamanhos

## 🚀 Como Usar

1. Acesse o app via navegador
2. Clique em **"➕ Adicionar paciente"**
3. Preencha os dados:
   - Nome do paciente
   - Contato (telefone/email)
   - Queixa/Descrição
4. Clique em **"Salvar"**
5. Os dados aparecem na lista abaixo

## 💾 Instalação como App

### Android
1. Abra o app no Chrome
2. Toque no menu (⋮)
3. Selecione **"Instalar app"**

### iOS
1. Abra o app no Safari
2. Toque em **Compartilhar**
3. Selecione **"Adicionar à Tela de Início"**

### Desktop (Windows/Mac/Linux)
1. Abra o app no Chrome/Edge
2. Clique no ícone **Instalar** na barra de endereço
3. Confirme a instalação

## 📁 Arquivos do Projeto

- **index.html** - Interface principal do app
- **manifest.json** - Configuração do PWA
- **service-worker.js** - Cache e suporte offline

## 🔧 Tecnologias

- HTML5

- PWA (Progressive Web App)
